# newsapp

날씨와 미세먼지 데이터를 받아와서 한줄 알림을 해주는 앱입니다.
ex) 4월 20일 수요일입니다. 날씨가 춥군요 오늘 날씨는 
    맑고 미세먼지 농도는 높음입니다. 초미세먼지 농도는 보통입니다.
